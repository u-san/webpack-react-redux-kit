# webpack-react-redux-kit

## 1.1 start

```
$ npm install
```
## 1.2 dev
```
$ npm run dev
```

查看效果` http://127.0.0.1:8080`

## 1.3 build
```
$ npm run build
```

## 技术栈

- [x] [Webpack](https://webpack.github.io)
